# Discord.JS v14 Bot Template

## Requirements

1. NodeJS latest LTS version (This will come bundled with NPM)

## How to use
1. Check that nodeJS is installed correctly by doing `node -v` in a terminal, if you get an error try restarting your computer
2. Cd into the project by doing `cd .\Discord.JS-v14-typescript-bot-template\`
3. Install dependencies `npm install`
4. Put your bot's token on the `config.json` file, if you dont have a bot go to the [discord application page](https://discord.com/developers/applications) and create one
5. Compile project by doing `npm run build`
6. Deploy commands by doing:
    * For global: `npm run deploy-global`
    * For local (guild only) `npm run deploy-guild`
7. Run the project by doing `npm run dev`
